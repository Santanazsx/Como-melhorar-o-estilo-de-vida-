# 🍏 Como melhorar o estilo de vida

Um plano simples para sair do sedentarismo. O foco é trocar hábitos ruins por 3 metas reais:
- 💧 Beber mais água
- 👟 Caminhar 15 minutos
- 😴 Dormir melhor

O objetivo é ter mais energia no dia a dia sem complicar a rotina. É o básico que funciona para quem está começando agora.
